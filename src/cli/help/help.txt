**help** | View help on a specific FluentCV command

Usage:

   **fluentcv HELP [<command>]**

   The HELP command displays help information for a specific
   FluentCV command, including the HELP command itself.

Parameters:

   **<command>**

      The FluentCV command to view help information for.
      Must be BUILD, NEW, CONVERT, ANALYZE, VALIDATE, PEEK,
      or HELP.

         fluentcv help convert
         fluentcv help help

Options:

   **None.**
